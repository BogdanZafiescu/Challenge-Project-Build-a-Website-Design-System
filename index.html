Colors

Ocen BLue

Hex: #1D64F2

RGB: rgb(29,100,242)

Ocean Blue is intended to be used with borders or shadows.

Shallow BLue

Hex: #1A80D9

RGB: rgb(26,128,217)

If we are going to create banners or navigation menus, we are going to use this color as the background.

Sky BLue

Hex:#1AB6D9

RGB:rgb(26, 182, 217);

Sky Blue should be used as a background for the body of the pages or it can be used as background for the active and hover states of links or buttons;

Green Water

Hex:#1AB6D9

RGB:rgb(26, 182, 217);

We are going to use this color for any lynk's active and hover state.

Bad Memories

Hex:#F2C12E

RGB:rgb(242, 193, 46);

This color is for titles, subtitles, buttons or links.
